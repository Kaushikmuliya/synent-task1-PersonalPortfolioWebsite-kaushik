# Personal Portfolio Website
https://kaushikmuliya.github.io/synent-task1-PersonalPortfolioWebsite-kaushik/

A single-page personal portfolio built with plain HTML, CSS, and JavaScript. The design uses a dark, neon-inspired aesthetic and includes small interactive details (custom cursor, reveal animations) while remaining framework-free.

## Features

- Hero section with introduction and CTAs
- Sections: About, Skills, Projects, Experience, Achievements, Certifications, Contact
- Lightweight UI animations: custom cursor, scroll hint, section reveal
- Responsive layout and mobile navigation
- No build step required; open the HTML file in a browser to view

## Project Structure

- `index.html` — Main portfolio page (HTML markup)
- `style.css` — Extracted stylesheet (visuals and layout)
- `README.md` — Project overview and usage

## Quick start

1. Open `main.html` in your browser to preview.
2. Edit `main.html` and `style.css` to customize content and styles.

## Notes

- This repository is intentionally minimal so it can be hosted directly (GitHub Pages, static host) or expanded into a small project with separate JS/CSS files.
