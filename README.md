# Personal Portfolio Website

A bold single-page personal portfolio built with plain HTML, CSS, and JavaScript. The page uses a dark neon visual style with a custom cursor, smooth scrolling, section-based navigation, and responsive layouts for desktop and mobile.

## Features

- Hero section with intro, call-to-action buttons, and live stats
- About, skills, projects, experience, achievements, certifications, and contact sections
- Animated UI details including a custom cursor, scroll hint, and hover effects
- Responsive navigation with a mobile menu
- No framework or build step required

## Project Structure

- `index.html` - Main portfolio page containing the markup, styles, and scripts
